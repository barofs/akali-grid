Semantic

Classes on akali grid are semantic, which improves your code's readability, maintanability and consequently your team's productive.

Because of it sounds like sentences, classes like "four wide column" are easier for everyone to understand what's happening on the front page, even when you're not a front-end developer.

Take a look at how semantic your HTML will become with Akali

Responsive

Akali is mobile-first. So when you set a column to be 'twelve wide', it will start by styling the mobile and automatically will maintain the same size on other devices.

You can set columns wides on each device if necessary. test

Magical

You'll love Akali, because it comes with unbelievable handy utilities.

  • Easily fill columns spaces with 'fill column', no math :-P
  • Easily center grids with 'centered grid'
  • Create columns that will take full width on tablet and mobile no matter what with 'stackable grid'
  • Determine which device the column should appear by using 'computer only', etc
  • Set offset for each device 'offset two on tablet' or set global offset 'offset two column'
  • Reverse a grid order with 'reversed grid', if grid is stackable, 'stackable reversed grid'
